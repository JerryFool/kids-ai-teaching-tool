- 首页：AI未来家长课堂项目入口

- 第1课：未来考古，看懂AI时代变局

封面图模板规则：

- 比例：16:9 横版。
- 主视觉：一个明确课堂主题场景，不堆太多元素。
- 角色：原创蓝白机器人老师 + 1-2 个孩子。
- 画面语言：用图标、道具、动作表达课程主题，尽量不在图片里放文字。
- 风格：明亮蓝白、轻赛博、儿童友好、克制高级。
- 课内插图规则：每张图必须服务一个教学动作，例如讲故事、做对比、引导填写，不做纯装饰图。
