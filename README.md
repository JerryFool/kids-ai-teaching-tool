# 孩子AI互动教学工具

这是 Jerry 给两个孩子讲 AI 家庭课的网页课件样板。

当前版本：

- 首页：AI未来家长课堂项目入口
- 第一阶段：只做第5课高质量样板
- 第5课：AI创意画室，把脑袋里的画说清楚
- 使用场景：iPad 横屏、电脑投屏
- 形式：网页互动课程站
- 品牌：思美奇主理人 Jerry Fu / CyberMagic 作品
- 体验：原创蓝白轻赛博卡通风、每页不同课堂视觉、轻动画、音乐开关、声效反馈、家长讲义、每页 Tips、English Practice
- 音乐：本地 MP3 轻音乐，来源 Orange Free Sounds，CC BY 4.0

说明：

- 第5课以“训练孩子把想象画面说清楚”为核心目标。
- 当前互动包括：描述拼图、模糊句升级、AI画错修正、孩子设计卡、双人任务卡。
- 当前视觉升级包括：首页未来学校场景、每页不同代码原生插画、进度条、触控按钮和更适合 iPad 的字号排布。
- 首页风格样板图：`assets/images/home-hero-future-school-v1.png`，用于确定后续课程封面和课内插图的视觉方向。
- 等第5课样板确认后，再批量扩展31课完整目录和内容。
- 工具重点不是让孩子直接依赖 AI，而是训练观察、描述、修改和表达。
- 当前二维码只放在首页品牌区，不放入每课总结页。
- 音乐文件：`assets/music/calm-piano-background.mp3`
